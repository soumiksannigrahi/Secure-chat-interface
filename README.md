# 🔐 Secure Chat Interface (CipherChat)

A browser-based real-time chat application with **End-to-End Encryption (E2EE)** using Web Crypto API.

## 🚀 Features
- AES-GCM encryption (secure messaging)
- PBKDF2 key derivation
- Real-time communication via MQTT (WebSockets)
- Multi-user support (up to 10 users)
- Shareable encrypted room links
- Responsive modern UI

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Web Crypto API
- MQTT.js (HiveMQ broker)

## ▶️ Run
Open `index.html` in your browser.
